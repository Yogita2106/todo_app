
# 🚀TODO App with Auth

Ek modern, fully functional TODO application jise **MERN Stack** (MongoDB, Express, React, Node.js) ka use karke banaya gaya hai. Isme **JWT Authentication** aur **Cloud MongoDB** integration hai.

## ✨ Features
* **User Auth:** Secure Login aur Register (bcrypt password hashing).
* **CRUD Operations:** Tasks ko Create, Read, Update aur Delete karne ki suvidha.
* **Cloud Database:** MongoDB Atlas ka use kiya gaya hai data persistence ke liye.
* **Modern UI:** Glassmorphism design aur dark theme.
* **Responsive:** Mobile aur Desktop dono par mast chalta hai.

## 🛠️ Tech Stack
* **Frontend:** React.js, Axios, CSS3
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Security:** JWT (JSON Web Tokens), Bcryptjs
